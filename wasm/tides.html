Method

Presentation

A signal can be approximated with the following series, \[ h(t) = H_0 + \sum_{j = 1}^{n}{H_j f_j \cos(a_j t + \theta_j)} \] In the case of tides, the pulsations $a_j$, and the node factor $f_j$ are known astronomical constants. $f_j$ should be close to 1 and vary very slowly over years.

We choose to write $A_j = H_j f_j$ when approximating $f_j$ as constant over time.

Problem statement

We want to find the $2 \cdot n$ parameters $\theta_j$ and $A_j$ such that they minimise the residual $r_i$ for the $m$ data points $(t_i, h_i)$ . \[ h_i - H_0 - \sum_{j}^{n}{H_j f_j \cos(a_j t_i + \theta_j)} = r_i \] \[ h_i - f_i(H, \theta) = r_i \] $f_i$ is not a linear operator since $\theta$ is in the cosinus, however, the original series can be rewritten to get a linear optimization problem.

Linearisation

Using the angle sum identity, one can show that, with well chosen $c_j$ and $s_j$, \begin{equation*} \begin{split} h(t) &= H_0 + \sum_{j}{A_j \cos(a_j t + \theta_j)} \\ &= H_0 + \sum_{j}{c_j \cos(a_j t)} + \sum_{j}{s_j \sin(a_j t)} \end{split} \end{equation*} Thus the problem can be written using a linear operator $\mathbf{F}$, \[ \mathbf{h} - \mathbf{F} \cdot \mathbf{x} = \mathbf{r} \]

The relations between the parameters {$A_j$, $\theta_j$} and {$c_j$, $s_j$} are retrieved as follow, \[ c_j \cos(a_j t) + s_j \sin(a_j t) = \sqrt{{c_j}^2 + {s_j}^2 } \left[ \hat{c}_j \cos(a_j t) + \hat{s}_j \sin(a_j t) \right] \] with, $ \hat{c}_j = \frac{c_j}{\sqrt{{c_j}^2 + {s_j}^2 }} $ and $ \hat{s}_j = \frac{s_j}{\sqrt{{c_j}^2 + {s_j}^2 }} $

Then, we can find $\theta$ such that, \begin{equation*} \begin{split} \hat{c}_j \cos(a_j t) + \hat{s}_j \sin(a_j t) &= \cos(\theta_j)\cos(a_j t) - \sin(\theta_j) \sin(a_j t) \\ &= \cos(a_j t + \theta_j) \end{split} \end{equation*}

Thus, \begin{equation*} \begin{split} \theta_j &= \arctan2(-s_j, c_j) \\ A_j &= \sqrt{{c_j}^2 + {s_j}^2 } \end{split} \end{equation*}
