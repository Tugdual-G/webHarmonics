Online Tide Least Square Harmonics Estimation

This online harmonic analyser can parse sea level data in text format to carry the estimation.

Each line in the file should correspond to a data point, at the exception of the eventual header or comments.

If the data is in datetime format, provide a format string according to link . Otherwise, if the times values are just plain numeric values, please provide the number of seconds per unit of time in the format field (ex: 3600 for hourly data).
